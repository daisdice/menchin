# 麻雀メンチン待ち当てクイズ

清一色（チンイツ）の多面張待ちを当てるクイズゲームです。

## 遊び方
1. `index.html` をブラウザで開きます。
2. 「ゲームスタート」ボタンを押します。
3. 表示された手牌の待ち（アガリ牌）をすべて選択します。
4. 「回答する」ボタンを押します。
5. 制限時間（60秒）内に何問正解できるか競います。

## デプロイ方法 (GitHub Pages)
このリポジトリの `Settings` -> `Pages` から、ソースを `main` ブランチ（または `master`）に設定し、フォルダを `/` (root) に設定して保存してください。
数分後に公開されます。

## 技術スタック
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- 外部ライブラリ不使用

## ファイル構成
- `index.html`: エントリーポイント
- `css/style.css`: スタイルシート
- `js/mahjong.js`: 麻雀ロジック（手牌生成、待ち判定）
- `js/game.js`: ゲーム進行管理
