# Project Overview: Menchin

## Description
Menchin is a quiz game for practicing Mahjong "Chinitsu" (Full Flush) multi-wait patterns.
Users identify wait tiles from a given hand to improve their Mahjong skills.

## Technology Stack
- **Core**: React 19, TypeScript, Vite
- **State Management**: Zustand
- **Routing**: React Router
- **UI/Styling**: Framer Motion, CSS Modules, CSS Variables
- **PWA**: vite-plugin-pwa
- **Deployment**: GitHub Pages

## Directory Structure
- `src/`: Source code
- `.agent/rules/`: Project-specific agent rules
